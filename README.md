# 소개
John Ahn님 강의를 보고 만든 youtube clone입니다.
### 기술 스택
- **React**  
React hooks를 이용하여 프론트엔드를 개발하였습니다.
- **Redux**  
Redux를 통해서 state를 관리하였습니다.
- **Express.js**  
node.js의 라이브러리인 Express.js를 이용하여 백엔드를 개발하였습니다.
- **MongoDB**  
DB로는 몽고디비를 사용하였습니다.
- **기타**  
  - Axios를 통해 ajax통신을 하였습니다.
  - multer을 통해 동영상 파일을 처리하였습니다.
